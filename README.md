# ST558_FinalProject

Packages needed to run app:
 - Shiny
 - Tidyverse
  
  
Code to install necessary packages:
```r
install.packages("shiny")
install.packages("tidyverse")
```
  
  
Code to run app:
```r
shiny::runGitHub(repo="ST558_FinalProject",
                 username="mdolan77",
                 subdir="ST558_Final")
```
